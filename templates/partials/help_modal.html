Help & Tips

1. Upload a Dataset

  • Drag & drop or click to browse a .csv, .xlsx, or .xls file.
  • The first row must contain column headers.
  • The dataset schema is detected automatically.

2. Example Questions

  • show female patients older than 30
  • average salary by department
  • top 5 highest salary
  • count students from Dhaka
  • total sales by region
  • age between 20 and 40

3. Supported Keywords

  • average, avg → AVG()
  • total, sum → SUM()
  • count, how many → COUNT()
  • maximum, max → MAX()
  • minimum, min → MIN()
  • highest → ORDER BY DESC
  • lowest → ORDER BY ASC
  • top N → LIMIT
  • by → GROUP BY
  • greater than, more than → >
  • less than, below → <
  • between → BETWEEN
  • is, equal to → =

4. Tips for Best Results

  1. Use simple English questions.
  2. Mention column names whenever possible.
  3. Use by <column> for grouped results.
  4. Use top N or lowest N for ranking.
  5. Review the generated SQL before execution.

5. Current Limitations

  • English language only.
  • Single-table queries only.
  • OR conditions are not supported.
  • JOIN queries are not supported.
  • Natural language dates (e.g. last month) are not supported.

If a question is unclear or does not match the uploaded dataset, the app will display an error instead of generating an incorrect SQL query.
