Help & Tips

1. Upload a Dataset

  • Drag & drop or click to browse a .csv, .xlsx, or .xls file.
  • The first row must contain column headers.
  • The dataset schema is detected automatically after upload.

2. Example Questions

  • show female patients older than 30
  • list all students
  • average salary by department
  • top 5 highest salary
  • lowest 10 prices
  • count students from Dhaka
  • how many female employees
  • total sales by region
  • maximum age
  • minimum salary
  • age between 20 and 40
  • employees with salary greater than 50000

3. Supported Keywords

  • average, avg → AVG()
  • total, sum → SUM()
  • count, how many, number of → COUNT()
  • maximum, max → MAX()
  • minimum, min → MIN()
  • highest → ORDER BY DESC
  • lowest → ORDER BY ASC
  • top N → ORDER BY DESC LIMIT N
  • first N → LIMIT N
  • last N → ORDER BY DESC LIMIT N
  • by → GROUP BY
  • greater than, more than, above → >
  • less than, below, under → <
  • between → BETWEEN
  • is, equals, equal to → =

4. Accepted Question Starters

  • show, list, display, find, get, fetch, retrieve, return, select
  • what, which, who, where, when, how
  • please and kindly are optional.

5. Tips for Best Results

  1. Use simple English questions.
  2. Mention column names whenever possible.
  3. Use by <column> for grouped results.
  4. Use top N or lowest N for ranked results.
  5. Use greater than, less than, or between for filtering.
  6. Review the generated SQL before execution.

6. Current Limitations

  • English language only.
  • Supports single-table queries only.
  • OR conditions are not supported.
  • JOIN queries are not supported.
  • Natural language dates (e.g. last month) are not supported.

If a question is unclear or does not match the uploaded dataset, the system will display an error instead of generating an incorrect SQL query.
