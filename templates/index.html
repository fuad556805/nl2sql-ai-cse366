About nl2sql

nl2sql is a hybrid natural-language-to-SQL tool. It combines a machine-learning intent classifier with rule-based attribute matching to turn plain-English questions into real SQL queries — no coding required.

  • Upload any CSV file — columns and sample values are read automatically.
  • Type a question like "average salary by district".
  • The AI detects your intent, matches the right columns, and builds a safe SQL query.
  • Results appear instantly. No data ever leaves this app — everything runs locally.

Help & Tips

1. Upload a dataset

  • Drag & drop or click to browse a .csv file.
  • The first row must contain column headers.

2. Ask a question

  • show female patients older than 30
  • average purchase amount by location
  • top 5 highest salary
  • count students where district is Dhaka

3. Tips for best results

  1. Mention the column name in your question (e.g. "purchase amount", "review score").
  2. Use average, total, maximum, minimum for aggregates.
  3. Use top N / lowest N for ranked results.
  4. Use by <column> to group results.

If a question is too ambiguous, the app will show a clear error instead of guessing.
